功能测试说明

本测试页面用于验证您的 Vue 应用通过 Service Worker 提供 accessToken 的功能是否正常工作：

  • 此页面模拟外部系统获取 accessToken 的场景
  • 它通过 fetch API 请求 /api/service-worker/getAccessToken 端点
  • 您的 Service Worker 应该拦截此请求并从 Pinia Store 获取数据
  • 下方控制面板允许您模拟各种测试场景
  • 测试前请确保您的 Vue 应用已在浏览器中运行
